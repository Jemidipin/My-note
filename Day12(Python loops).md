# Python loops
## For loop 
  #### for iterator_var in sequence:
  ####  statements(s)
## While loop
  while expression:
    statement(s)
### Syntax of While Loop with else statement:
   while condition:
     # execute these statements
   else:
     # execute these statements
## Nested loop
   for iterator_var in sequence:
      for iterator_var in sequence:
         statements(s)
   statements(s)
